# Delivered to You Parcel Service - Mock Tracking Website

A professional-looking mock parcel tracking website for demonstration purposes.

## Features

- Clean, modern user interface
- Real-time package tracking simulation
- Detailed tracking timeline with location updates
- Responsive design for mobile and desktop
- Multiple tracking scenarios

## Demo Tracking Numbers

Try these tracking numbers to see different delivery scenarios:

- **DT123456789** - In Transit from China
- **DT987654321** - Customs Clearance 
- **DT456789123** - Out for Delivery

## Shipment Details

All mock shipments show:
- **Origin:** China
- **Receiver:** Candice Sean
- **Delivery Address:** 1 Vale St, St Kilda VIC, Australia

## How to Use

1. Open `index.html` in your web browser
2. Enter one of the tracking numbers above
3. Click "Track Package" or press Enter
4. View the detailed tracking information and timeline

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- Vanilla JavaScript
- No external dependencies

## Setup

Simply open the `index.html` file in any modern web browser. No server or installation required!

## GitHub Pages

To host this site on GitHub Pages:

1. Go to repository Settings
2. Navigate to Pages section
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Save and wait a few minutes for deployment

Your site will be available at: `https://missdaze.github.io/Deliveredtoyouparcelservice/`
